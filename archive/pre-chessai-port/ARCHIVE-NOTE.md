# Pre-CHESSai-port archive

These files predate the 2026-07-03 CHESSai architecture port (see the repo's
`CLAUDE.md` §20) and describe architecture that no longer exists — the old
`src/coaches/*` persona files (`vlad.js`, `fabiano.jsx`, `hikaru.jsx`,
`magnus.jsx`), the old multi-page `App.jsx` module structure
(`GameAutopsy`, `DrillSergeant`, `OpeningLab`, `MiddlegameMat`,
`EndgameDojo`, `VideoLibrary`, `Dashboard`, `Profile`), and the old
`askCoach(persona, message)` mechanism.

They also predate the 2026-07-04 identity rename (`CLAUDE.md` §21) and
still refer to the app as "Vlad Chess Coach" and to the repo by its old
name/URL throughout.

Kept for historical record only — not maintained, not accurate to the
current app. `CLAUDE.md` at the repo root is the current source of truth.

- `VLAD.md` — original project bible
- `README.md` — original repo README
- `Prompt.md` — pasteable AI session-start prompts referencing the old docs
- `SESSION-LOG.md` — old automated session log
- `Coach-memory.json` — old per-weakness coach memory/scheduling data
- `gentlemans-assassin.json` — old structured opening knowledge base
